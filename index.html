Configurar conexión online

Este juego es un único HTML. Para que varias casas vean la misma partida necesita una base online. Firebase Realtime Database tiene plan gratuito y alcanza para jugar con familia. La configuración pública de Firebase no es una contraseña, pero no uses esto para apuestas ni partidas donde importe evitar trampas.
Crear proyecto Firebase: entra a Firebase Console, crea un proyecto y agrega una “Web App”. Copia el objeto firebaseConfig.
Activar Auth: Authentication → Sign-in method → Anonymous → Enable.
Activar Realtime Database: Realtime Database → Create Database. Reglas recomendadas para uso familiar:
{ "rules": { "rummyRoomsV1": { ".read": "auth != null", "$room": { ".write": "auth != null" } } } }
Consejo: subí este HTML a Netlify Drop, GitHub Pages o cualquier hosting estático. Luego copiás el link de invitación desde la sala.
